## 📖 项目简介

**LLM-100-Lessons** 是一个专注于大型语言模型（Large Language Model）与 AI Agent 技术的深度技术知识库。本仓库汇集了100+核心主题的深度分析，涵盖从模型预训练、微调技术到 Agent 系统架构的完整技术栈。

无论你是 AI 研究者、算法工程师，还是希望深入理解大模型技术的开发者，这里都能为你提供系统性的学习路径和实战参考。

## 🤝 贡献指南

欢迎提交 Pull Request 或 Issue！

### 贡献方式
1. **完善现有文章** - 补充最新论文、更新技术内容
2. **新增技术主题** - 提交新的深度分析文章
3. **修正错误** - 发现问题请提交 Issue
4. **翻译分享** - 将优秀内容翻译成其他语言

